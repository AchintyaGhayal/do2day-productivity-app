# Do2Day

## Overview
Do2Day is an iOS productivity and workflow management application designed to help students manage tasks, notes, deadlines, and scheduling through structured daily planning.

The platform integrates task management, calendar scheduling, and persistent data storage into a unified system that reduces friction in daily productivity workflows.

---

## Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS
- Supabase (PostgreSQL, Authentication)
- REST APIs
- Median.co (iOS deployment)

---

## System Features
- Automated time-blocking and task scheduling
- Integrated Inbox, Today, Calendar, and Notes workflow system
- Persistent user data storage using Supabase
- Real-time task updates and synchronization across sessions
- Structured daily planning with dynamic scheduling logic

---

## Architecture

- **Frontend:** Built using Next.js and React for modular UI components and dynamic task rendering  
- **Backend:** Supabase (PostgreSQL) used for structured data storage and real-time synchronization  
- **Authentication:** Supabase Auth manages secure user sessions and persistent login state  
- **Data Flow:** REST API-based communication between frontend and backend to manage tasks, scheduling, and notes  
- **Deployment:** Built using Lovable and deployed to iOS via Median.co  

---

- Designed modular task-management logic with dynamic scheduling algorithms
