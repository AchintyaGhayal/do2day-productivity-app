# Do2Day

## Overview
Do2Day is an iOS productivity and workflow management application designed to help students manage tasks, notes, deadlines, and scheduling through structured daily planning.

## Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS
- Supabase (PostgreSQL, Auth)
- Median.co (iOS deployment)

## Key Features
- Automated time-blocking and task scheduling
- Inbox, Today, Calendar, and Notes workflow system
- Persistent user data with Supabase backend
- Cross-session state synchronization

## Technical Highlights
- Built full-stack architecture using Supabase for authentication and database management
- Implemented real-time data synchronization across tasks and scheduling workflows
- Designed modular task logic with dynamic scheduling algorithms
- Structured frontend state management for consistent user experience
